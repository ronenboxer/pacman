Code for "cherry" for developers:

IconifyIcon web component makes using icons as easy as icon fonts. To use "cherry" in HTML, add this code to the document:

<iconify-icon icon="lucide:cherry" style="color: red;" width="30" height="30"></iconify-icon>

Web component will load icon data from Iconify API and render SVG.

Make sure you import IconifyIcon web component:

<script src="https://code.iconify.design/iconify-icon/1.0.0/iconify-icon.min.js"></script>

Web component works with almost all modern UI frameworks. Click documentation link below.

Click here for more information about IconifyIcon web component.

Data for icons is open source and is available for everyone to use. You can use it with various icon components, create your own components that use Iconify as source or just export raw SVGs.

All icons are validated, cleaned up and automatically kept up to date.
